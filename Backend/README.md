This project is a backend for a **Course Management System (CMS)**, built with Node.js, Express, and MongoDB.  
It includes **role-based access control** (Admin, Instructor, Student), **JWT authentication**, and full CRUD operations for courses.  

Features implemented:

- User registration and login
- Role-based access control (Admin / Instructor / Student)
- Course management (create, read, update, delete)
- Student enrollment
- View enrolled students (Instructor / Admin)
- JWT-protected routes
- Password hashing with bcrypt
- Clean folder structure and error handling
